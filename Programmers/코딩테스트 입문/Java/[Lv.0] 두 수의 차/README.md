# [Lv.0] 두 수의 차

<br>

## 문제설명
정수 `num1`과 `num2`가 주어질 때, `num1`에서 `num2`를 뺀 값을 return 하도록 solution 함수를 완성해 주세요.

<br>

## 제한사항
- -50,000 ≤ `num1` ≤ 50,000
- -50,000 ≤ `num2` ≤ 50,000

<br>

## 입출력 예
| num1 | num2 | result |
|---|---|---|
| 2 | 3 | -1 |
| 100 | 2 | 98 |

<br>

## 입출력 예 설명
- 입출력 예 #1
    - `num1`이 2이고, `num2`가 3이므로 2 - 3 = -1을 return 합니다.

- 입출력 예 #2
    - `num1`이 100이고, `num2`가 2이므로 100 - 2 = 98을 return 합니다.
