# [Lv.0] 자릿수 더하기

<br>

## 문제설명
정수 `n`이 매개변수로 주어질 때, `n`의 각 자리 숫자의 합을 return 하도록 solution 함수를 완성해 주세요.

<br>

## 제한사항
- 0 ≤ `n` ≤ 1,000,000

<br>

## 입출력 예
| n | result |
|---|---|
| 1234 | 10 |
| 930211 | 16 |

<br>

## 입출력 예 설명
- 입출력 예 #1
    - 1 + 2 + 3 + 4 = 10을 return 합니다.

- 입출력 예 #2
    - 9 + 3 + 0 + 2 + 1 + 1 = 16을 return 합니다.
