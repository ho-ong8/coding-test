# 홀짝 구분하기

<br>

## 문제설명
자연수 `n`이 입력으로 주어졌을 때, 만약 `n`이 짝수이면 "`n` is even"을, 홀수이면 "`n` is odd"를 출력하는 코드를 작성해 보세요.

<br>

## 제한사항
- 1 ≤ `n` ≤ 1,000

<br>

## 입출력 예
- 입력 #1
    ```java
    100
    ```

- 출력 #1
    ```java
    100 is even
    ```

- 입력 #2
    ```java
    1
    ```

- 출력 #2
    ```java
    1 is odd
    ```
