# 문자열로 변환

<br>

## 문제설명
정수 `n`이 주어질 때, `n`을 문자열로 변환하여 return 하도록 solution 함수를 완성해 주세요.

<br>

## 제한사항
- 1 ≤ `n` ≤ 10,000

<br>

## 입출력 예
| n | result |
|---|---|
| 123 | "123" |
| 2573 | "2573" |

<br>

## 입출력 예 설명
- 입출력 예 #1
    - 123을 문자열로 변환한 "123"을 return 합니다.

- 입출력 예 #2
    - 2573을 문자열로 변환한 "2573"을 return 합니다.
