# 문자열 돌리기

<br>

## 문제설명
문자열 `str`이 주어집니다.<br>
문자열을 시계방향으로 90도 돌려서 아래 입출력 예와 같이 출력하는 코드를 작성해 보세요.

<br>

## 제한사항
- 1 ≤ `str`의 길이 ≤ 10

<br>

## 입출력 예
- 입력 #1
    ```java
    abcde
    ```

- 출력 #1
    ```java
    a
    b
    c
    d
    e
    ```
